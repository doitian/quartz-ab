# About

This is an example Obsidian vault set up with Quartz for publishing to GitHub Pages.

## What is Quartz?

Quartz is a fast, batteries-included static site generator designed for hosting your digital garden and networked thought online.

## Features

- Fast and responsive
- Backlinks and graph view
- Full-text search
- Dark mode
- Mobile-friendly
